<h2>Instalação</h2><br/>
A aplicação foi criada com react. Para funcionar a aplicação.<br/>
1° passo:
<br/>
$ git clone https://github.com/silasms/front-registration
<br/>
2° passo:
<br/>
$ npm start
<br/>
Simplesmente assim e a sua aplicação já está rodando.
